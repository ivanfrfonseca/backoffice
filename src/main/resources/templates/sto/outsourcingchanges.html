Suporte / Outsourcing Changes

O que é Outsourcing Changes?

É um processo automático de alterações diversas.

Dessa forma, algumas atividades podem ser executadas de forma mais prática, sem a necessidade de ficar acessando várias tabelas no banco de dados.

Atividades como: Excluir um produto, alterar o identificador de um ponto de acesso, desfazer a desativação de um ponto de acesso e seus produtos, entre outros.

São cerca 40 funcionalidades que podem ser executadas de forma automática.

Este recurso foi desenvolvido pelo Paulo Afonso (afonso@cpqd.com.br).

Como utilizar?

A utilização é simples, é preciso fazer um insert na tabela "tbl_outsourcing_changes", seguindo as regras descritas na documentação.

Para processamento, o registro deve estar com status “N” e deve-se executar a procedure “run” da package “pck_outsourcing_changes”.

Ex.: begin pck_outsourcing_changes.run; end;
